CSS Advanced Project
Project Description
This project is the continuation of the HTML Advanced project. The focus is on implementing CSS styling to create a visually appealing webpage based on a Figma design mockup. This project transforms a basic HTML structure into a fully styled, responsive website using advanced CSS techniques.

Create a pixel-perfect implementation of the provided Figma design
Use only HTML and CSS (no external frameworks)
Implement responsive design principles
Follow semantic HTML best practices
Create clean, maintainable CSS code

Development Guidelines
HTML Standards

Use semantic HTML5 elements
Maintain proper document structure
Include alt attributes for images
Follow accessibility best practices

CSS Standards

Use class-based selectors
Organize CSS logically by sections
Implement responsive design with media queries
Avoid using !important declarations
